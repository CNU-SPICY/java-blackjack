## Extra Game Rule

- 딜러는 52장으로 이루어진 카드 덱을 게임 규칙에 따라 분배한다.
- 모든 카드는 중복되지 않는다.

- 에이스의 점수는 현재 점수에 따라 결정된다.
- 만약 21점보다 크다면, 21점을 넘지 않도록 최소한의 에이스만 1점으로 한다. 

- 플레이어는 21점보다 작을 때만 카드를 더 받을 지 선택할 수 있다.
